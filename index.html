Nuestros Comunicados

Nuestra organización cuenta con un servicio, ya sea presencial o a distancia, que tiene el fin de brindar una asistencia técnica útil y profesional que pueda suplir con las necesidades de nuestros clientes.

Los servicios que ofrecemos varian dependiendo si requieren realizarse de manera presencial o si es posible a distancia

  • Presencial

    • 1- Limpieza de Harware
    2- Cambio de componentes dañados
    3- Análisis de funcionamiento
    4- Actualización de Hardware

  • Distancia

    • 1- Actualización de Sistema Operativo
    2- Actualización de Aplicaciones
    3- Soporte sobre el uso de Aplicaciones
    4- Errores de software y/o Hardware de facil correción

  • Mixto

    • 1- Mantenimiento Preventivo
    Este tipo de mantenimientos se pueden programar para realizarse cada 3, 6, 8 u 12 meses
    2- Mantenimiento Correctivo
    3- Análisis de amenazas de software
    4- Actualización de Software
